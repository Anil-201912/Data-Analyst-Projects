# 📊 Retail Sales Analytics Dashboard

## 📌 Overview

This project analyzes retail sales data to provide insights into sales performance, profit, customer behavior, and product trends. The dashboard helps businesses make data-driven decisions by visualizing key performance indicators.

---

## 🎯 Objectives

- Analyze overall sales performance
- Track profit and revenue trends
- Identify top-performing products and categories
- Monitor regional sales performance
- Discover customer purchasing patterns

---

## 🛠️ Tools Used

- 📊 Microsoft Power BI
- 📈 Microsoft Excel
- 🗄️ SQL (Data Analysis)

---

## 📂 Project Files

- Retail_Sales_Raw.xlsx
- Retail_Sales_Cleaned_Data.xlsx
- Retail_Sales_Analytics_Dashboard.pbix
- Retail_Sales_Analytics_Dashboard.pdf

---

## 📈 Dashboard Features

- Total Sales
- Total Profit
- Total Orders
- Sales by Category
- Sales by Region
- Monthly Sales Trend
- Top Products
- Customer Insights

---

## 💡 Key Insights

- Identified the best-performing product categories.
- Analyzed monthly sales and profit trends.
- Compared sales performance across different regions.
- Highlighted business opportunities using interactive visualizations.
